  • 你好！我是 Jover，一个很菜的 Linux 爱好者。
  • 我对操作系统和硬件相关的东西比较感兴趣，也因此，我曾编写过一个玩具操作系统 DeliciOS (opens new window)。
  • 我好像对能跑起来的东西都比较感兴趣，所以我还编写了一个玩具编程语言 Water (opens new window)。
  • 我比较热衷于 Unix 艺术，这使我对命令行的世界很沉迷（也使我喜欢做一些没什么用的事情）。
    • 这是我的 dotfiles (opens new window) 配置，如果你跟我一样喜欢折腾这些没什么用的东西，这或许会对你有所帮助。
    • 同时我也深受 Vim 的毒害，这是我的 neovim-config (opens new window)。
  • 我曾经在我的 Linux 上编写过很多没什么用的工具，但这两个或许还有点用：
    • h (opens new window) 是受 asynctasks.vim (opens new window) 启发而开发的。h 是一个命令备忘录，能够通过模糊搜索来查找常用的命令和参数，非常适用于需要频繁执行一些命令的场景。
    • zsh-file-manager (opens new window) 是一个基于 zsh 的轻量级文件管理工具。其操作界面与 ranger (opens new window) 相似，但它更侧重于目录探索和文件预览，因此在目录间频繁穿梭时使用 zsh-file-manager 会更加高效。
  • 我总是需要动手实现一个东西，我才能够理解它。比如，只有我在动手实现了简单的 JIT 和 GC 之后，才觉得我好像有点理解它们了。
  • 我喜欢编程，喜欢哲学，也喜欢艺术。我的爱好广泛，欢迎交流和探讨。
